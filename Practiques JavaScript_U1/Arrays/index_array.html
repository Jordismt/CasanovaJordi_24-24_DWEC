INICI

Activitats UD1: Arrays JS

Exercici 1: matriu de 3 files i 5 columnes. El contingut de les cel·les ha de ser un indicador de la fila i la columna de manera que la primera fila quede f1c1, f1c2, f1c3... Exercici 2: Repeteix l'exercici anterior però has de crear la matriu a partir de tres files i afegir-les una a una fent ús del mètode push(). Exercici 3: Genera un array de 100 números aleatoris sencers entre 0 i 9. Conta el número de vegades que apareix cadascun dels dígits i mostra-ho per consola. Exercici 4: Genera un array de 25 números aleatoris sencers entre 0 i 100. Cap número es pot repetir. Mostra el resultat per consola. Exercici 5: Genera un array de 100 números aleatoris sencers entre 0 i 9. Fes que siguen 0 totes les caselles des de la 25 fins a la 75. Exercici 6: Genera un vector amb noms d'animals. Demana per pantalla a l'usuari que introduisca un nom i determina si existeix al vector o no amb includes(). Exercici 7: A partir d'un vector de paraules genera un string amb totes les paraules unides. Exercici 8: Ordena alfabèticament un vector de noms. Exercici 9: Fes un vector de noms. Amb filter() torna un vector que tinga aquells noms amb 6 o més lletres. Exercici 10: Genera una matriu on cada fila serà el nom d'un producte i el seu preu en euros. Posa el preu entre 0 i 100€. Aplica les funcions necessàries per a, a partir d'aquesta matriu, obtindre altra matriu amb aquells productes que el seu preu siga major de 75€ i s'haja aplicat una rebaixa del 10%.
